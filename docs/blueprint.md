# **App Name**: Affluence Arena

## Core Features:

- Content Creation Suite: Intuitive editor with real-time SEO optimization tips and suggestions to improve content visibility.
- Affiliate Link Management: System to add, manage, and track performance of affiliate links within posts.
- Real-time Analytics: Dashboard to monitor views, clicks, conversions, and earnings per post, updated in real-time.
- Challenge System: Gamified challenge system with tiered rewards to motivate content creation and engagement, promoting both SEO and affiliate success.
- Post Ranking AI tool: An AI tool that analyses new user-submitted posts to make a relevance determination. Highly relevant posts receive a ranking boost and feature more prominently in feeds.
- User Content Feed: Main social feed that ranks posts via a reddit style approach where the more interaction a post gets, the more visible it is.

## Style Guidelines:

- Primary color: Saturated, vibrant green (#4CAF50) to reflect earnings, growth, and activity.
- Background color: Dark, desaturated gray (#333333) to provide a sophisticated, focused backdrop.
- Accent color: Analogous yellow-green (#8BC34A), used sparingly to draw attention to interactive elements and calls to action.
- Body and headline font: 'Inter', a grotesque sans-serif known for being modern, machined, objective, and neutral; 'Inter' will be used for both headlines and body.
- Use a set of consistent, modern icons (Material Design or similar) to represent actions, categories, and status within the platform.
- Clean, card-based layout that highlights content previews.
- Use subtle transitions and animations to provide feedback, reinforce interactions, and create a smoother user experience.
